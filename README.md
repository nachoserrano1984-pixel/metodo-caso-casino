Sistema de gestión de máquinas de casino en lenguaje C.
El programa permite:
- Registrar múltiples máquinas
- Almacenar datos mediante estructuras (struct)
- Calcular el promedio de ganancia por jugada
- Evaluar el rendimiento de cada máquina
- Generar un resumen general del sistema
- Desarrollado para la asignatura de Programación I.
